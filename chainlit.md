# 🔍 로그 분석 시스템에 오신 것을 환영합니다!

이 시스템은 AI를 활용하여 서버 로그를 자동으로 분석하고 문제를 진단합니다.

## 🚀 주요 기능

### 📊 자동 로그 분석
- **인프라 이슈**: 데이터베이스 연결 장애, 서버 오류, 메모리 누수
- **보안 위협**: XSS 공격, SQL Injection, 무차별 대입 공격
- **성능 문제**: N+1 쿼리, 느린 응답 시간, 리소스 병목

### 🎯 분석 과정

1. **로그 파싱**: PM2 로그 포맷 자동 인식
2. **카테고리 분류**: AI가 이슈 유형 자동 판별
3. **심층 분석**: 전문 분석 에이전트가 근본 원인 파악
4. **보고서 생성**: 실행 가능한 조치사항 제공

### 💡 사용 방법

1. 📎 **파일 업로드**: 클립 아이콘을 클릭하여 .log 파일 업로드
2. ⏳ **자동 분석**: AI가 자동으로 분석 시작
3. 📋 **결과 확인**: 상세 보고서와 권장 조치사항 확인
